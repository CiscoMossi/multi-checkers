	no back-end: Porjeto MultiCheckers.SignalR -> Program.cs -> string url = "http://"seu ip aqui":8082"; não pode ser localhost pq depois vai dar pau se tiver dois usuários de locais diferentes

	no front-end: jogo -> jogo.service.js -> connection = $.hubConnection('http://"mesmo ip de antes":8082/signalr');
